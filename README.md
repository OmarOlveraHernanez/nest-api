#Teslo APi
1. clonar proyecto
2. ``` yarn install ```
3. Clonar el archivo ```.env.template``` r renombrarlo ```.env```
4. cambiar las variables del dentorno
5. Levantar la db

```
docker-compose up -d
```

6. Ejecutar SEED
```
localhost:3000/api/seed
```
6. Levantar: ```yarn start:dev```
